## 開始

要在本地運行此網頁，請按照以下步驟操作：

1. 複製檔案：
```bash
git clone https://github.com/ckmsc39th/ckmsc39th.github.io.git
```
2. 安裝依賴：
```bash
yarn
```
3. 啟動伺服器：
```bash
yarn dev
```
4. 在瀏覽器中打開`http://localhost:5173/` 以查看網頁。🌐👀
